### Quickstart
1. Clone the repository
```git clone https://github.com/TheUnicat/krypton-llm-ui.git ```
2. Change directory
```cd krypton-llm-ui```
2. Install dependencies
```pip3 install -r requirements.txt```
3. Run the application
```python3 main.py```
4. Open the browser and navigate to http://127.0.0.1:8080
5. Get an Anthropic API key (USD $5 free credit) from https://console.anthropic.com
6. Click on the settings icon on the bottom left of the webui, with the profile picture and username, and click on "Settings" from the popup
7. Enter your Anthropic API key in the Anthropic api key field and click on "Save"
8. You can now start using the webui to interact with the Anthropic API, including Claude Opus, a model with close to GPT-4 performance
